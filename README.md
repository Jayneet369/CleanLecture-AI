## ✨ What is CleanLecture AI?

CleanLecture AI is a smart lecture-cleaning system that removes non-academic fluff from YouTube lectures using AI-generated timestamps.

It feels almost like magic.

Paste a lecture URL, generate fluff timestamps using the custom AI Gem, and CleanLecture automatically skips:
- greetings
- chat reading
- repeated filler
- unrelated jokes
- admin talk
- “samajh aaya?” loops
- and other non-study distractions

The result?

A clean, focused learning experience with only the actual academic content.

## 🧠 How It Works

### Step 1
Copy the YouTube lecture URL.

### Step 2
Open the custom AI Gem:

👉 https://gemini.google.com/gem/1kyGJb7aGdWaQF-sXMLJPx5n0e8n4thkG?usp=sharing

### Step 3
Select mode carefully:

- **Pro (free)** → Most accurate and highly recommended for long lectures (1.5hr+)  
- **Thinking mode** → Faster and still good, but accuracy is not guaranteed. Best for lectures up to around 1 hour  
- **Fast mode** → Not recommended

### Step 4
Paste the lecture URL in the Gem.

### Step 5
Copy the fluff timestamps generated by the Gem.

Example:
```text
00:27–01:04
02:09–03:08
05:14–05:52
```

### Step 6
Paste:
- the YouTube URL
- the timestamps

into CleanLecture.

Boom — distractions gone, productivity boosted, time saved.
